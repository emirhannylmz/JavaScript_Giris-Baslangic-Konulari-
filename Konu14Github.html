Git ve Github kullanımı

Git : versiyon kontrol sistemidir

GitHub : git sistemini kuullanrak projerleriizi yönetip depolayan uygulamadır.

Önemli git terimleri

Repository Kısaca repo, dosyalarınızın bulunduğu projeler
Master Projelerimizin karaları sürümü, ana gövdesi
Branch Git, ağaç mantığı ile çalışır ve proje birden fazla dala ayrılıp
Merge Branch'lerdeki değişikliklerin msater'da birleştirilme işlemidir.
Push Bilgisayardaki kodlarınızı GitHub'daki repoya gönderir
Pull GitHub'daki projemizin son halini bilgisayara çekmemizi sağlar
.gitignore Git'in takip etmesini istediğimz dosyaları (lokalde denediğimiz resimlervb)push işleminde repoya gönderilmesini engellememizi sağlar
Untracked(U: izlenmeyen) Git tarafından henüz izlenmeyen yeni eklenmiş dosyalar
Unstaged(Hazırlanmamış) Güncellenmiş fakat commitlenmek için hazırlanmamış dosyalara denir
Staged(Hazırlanmmış) Güncellenmiş ve commitlenmek için hazır dosyalar
Deleted Projeden silinmiş fakat Git üzerinden kaldırılmamış dosyalara denir
